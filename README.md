Design: https://www.frontendmentor.io/challenges/audiophile-ecommerce-website-C8cuSd_wx

## Run Project
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
